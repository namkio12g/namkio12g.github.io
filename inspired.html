Inspired - youtube clone

Inspired by YouTube, this project is a full-featured video-sharing platform designed to deliver a similar user experience. It enables users to browse, watch, and upload videos seamlessly

Key components

Authentication & Security

  • Google OAuth2 and JWT: Used Google OAuth2 for secure user login, along with JWT tokens for session management and access control, ensuring secure, token would be stored in cookies

Microservices Architecture

  • Microservices with RabbitMQ: Split core functions (video processing, authentication, and data management) into distinct services, coordinated with RabbitMQ for efficient task distribution and communication.
  • Video service (Port 2020):is responsible for creating and editing videos, transferring video files to the upload service for processing and uploading to the cloud. It retrieves videos from database through an API to push them to the front end using infinite scroll.
  • Channel service (Port 2021): manages all channel-related data and functionality, including user authentication and channel data organization.
  • Upload service (Port 2024):Provided functionalities for processing video files and thumbnails and uploading them to the cloud. It performs tasks such as transcoding videos into multiple resolutions, creating thumbnails, and efficiently managing cloud uploads.
  • Interation service (Port 2023):Manages user interactions, specifically comments on videos. It allows users to add, update, delete, and retrieve comments, supporting smooth real-time engagement on the platform.

Video Processing & Storage

  • FFmpeg for Video Resolutions Used: FFmpeg to create multiple video resolutions and thumbnails, optimizing playback across different devices and network conditions.
  • AWS S3 Integration: Leveraged AWS S3 for reliable, scalable cloud storage of video files, allowing high-speed content delivery and cost-effective storage solutions.

Frontend User Experience

  • React with Responsive UI: Developed an intuitive front end with React.js, SCSS, and Bootstrap, creating responsive and user-friendly interfaces for video interaction

Techonologies used

  • Front-End: HTML, Scss, Javascript (ReactJS), Bootstrap 5
  • Back-End: Node.js (Express), RabbitMQ, FFmpeg, Google OAuth2
  • Database: MongoDB, Cloud service (AWS service)
  • Unit test: PostMan
